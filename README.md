\# 🏦 Loan Approval Prediction System



A machine learning-based web application that predicts loan approval status based on applicant information with \*\*82.93% accuracy\*\*.



\## 📋 Project Overview



This project implements a complete end-to-end machine learning pipeline for predicting loan approvals. It includes:

\- Data preprocessing and cleaning

\- Exploratory data analysis

\- Model training with multiple algorithms (Decision Tree, Random Forest, XGBoost)

\- Web interface for predictions

\- REST API for integration



\## 🚀 Features



\- ✅ User-friendly web interface

\- ✅ Real-time predictions

\- ✅ Probability scores with visual indicators

\- ✅ 3 ML models compared (Decision Tree, Random Forest, XGBoost)

\- ✅ Best model: Random Forest (82.93% accuracy)

\- ✅ REST API endpoint



\## 🛠️ Tech Stack



\- \*\*Backend:\*\* Python, Flask

\- \*\*ML Libraries:\*\* Scikit-learn, XGBoost

\- \*\*Data Processing:\*\* Pandas, NumPy

\- \*\*Visualization:\*\* Matplotlib, Seaborn

\- \*\*Frontend:\*\* HTML, Bootstrap 5
